# XEDI Source Document Templates

These CSV, JSON and XML templates mirror the live xedi.ai docs at `/docs/examples/source-documents.json`.

Use them as starter export shapes for customer systems and as mapping context before creating workflows. The templates include the business-friendly fields and the required mapping fields for the supported document families:

- `order`: ORDERS, X12 850, TRADACOMS Order
- `order-change`: ORDCHG, X12 860
- `invoice`: INVOIC, X12 810, TRADACOMS Invoice
- `despatch-advice`: DESADV, X12 856, TRADACOMS ASN
- `sales-report`: SLSRPT, X12 852
- `inventory-report`: INVRPT, X12 846

Each document has `.csv`, `.json` and `.xml` variants.
